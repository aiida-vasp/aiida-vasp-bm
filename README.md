# aiida-vasp-bm

Bulk modulus workchain for AiiDA-VASP v2.0.0 or later

## Installation

```shell
git clone git@github.com:atztogo/aiida-vasp-bm.git
cd aiida-vasp-bm
pip install -e .
verdi daemon restart
```
